# ofxCeres on macOS with OF 0.11.0

A collection of openframeworks addons to make ofxCeres work on my current system: 
- macOS 10.15.7
- OF 0.11.0
- Xcode 12.4


# Installation

Use projectGenerator.

# Addons

ofxCeres
ofxAudioDecoder
ofxAssets
ofxLiquidEvent
ofxGrabCam
ofxCvGui
ofxSingleton
ofxTextInputField
